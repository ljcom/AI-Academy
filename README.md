# AI Academy Paper

Repositori ini berisi naskah paper **From Knowledge Engines to AI Academy Engines: Rethinking Conversational AI for Learning Progression** dalam format terstruktur per-bagian (`manuscript/sections`) dan versi gabungan PDF.

## Baca Naskah

- Versi sumber utama: [paper.md](paper.md)
- Versi terpisah per bagian: [manuscript/sections](manuscript/sections)
- Versi PDF hasil build: [build/ai-academy-split.pdf](build/ai-academy-split.pdf)

## Download PDF (raw.githubusercontent.com)

Gunakan format link berikut untuk download langsung PDF dari GitHub:

```text
https://raw.githubusercontent.com/<OWNER>/<REPO>/main/build/ai-academy-split.pdf
```

Contoh (ganti dengan repo Anda):

```text
https://raw.githubusercontent.com/your-username/AI-Academy/main/build/ai-academy-split.pdf
```

## Build PDF Lokal

```bash
./build.sh
```

Output default:

- `build/ai-academy-split.pdf`
